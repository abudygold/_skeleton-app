Accordion

Accordion is a component in the UI that functions to hide and display information. This accordion is made so that users don't open up too much content that confuses them. Basically this accordion has 2 important components, namely the title of the content and the content itself. So if a user wants to see information that only they wants to see, they can simply click on the accordion to view the information in it.
